# CleanWave Laundry Web Server

## Project Title
**Simple Web Server using Node.js**  
(Assignment Submission)

## About the Project
This is a basic web server built with Node.js using only the built-in `http` and `fs` modules.  
It serves a modern, fully designed **CleanWave Laundry** website with multiple pages and proper routing.

The server handles different routes, serves HTML pages, CSS file, and shows a custom 404 error page.

## Features
- Clean and modern responsive design
- Separate `styles.css` file (modular code)
- Beautiful Tailwind CSS + imgage
- Proper navigation bar on every page
- 4 different pages
- Custom 404 error page
- 
- Easy to run and test

## Pages & Routes

 Page Name       \ Description                     
-----------------/---------------------------------/------
 Home            / Hero section + services preview /
 About Us        / Company story and features      /
 Contact         / Contact form and details        /
 Services        / Detailed service list           /
 404 Error       / Custom not-found page           /

## How to Run the Project

1. Make sure you have **Node.js** installed on your computer.
2. Open the project folder in terminal / command prompt.
3. Run the following command:

 in bash
 ------------
   node server.js
